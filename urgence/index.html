HTML Form Reference

Form Tag

Tag Name Description
<form> form the tag is used to create a form.
<input> input The tag is used to create an input element.
<label> label The tag is used to create a label for an input element.
<select> select The tag is used to create a select element.
<option> option The tag is used to create an option element for a select element.
<textarea> textarea The tag is used to create a button element.
<fieldset> fieldset The tag is used to group related elements in a form.
<button> button The tag tag defines a clickable button.

Attribute used

Attribut Description
action Spécifie l'URL à laquelle les données du formulaire seront soumises.
method Specifies the method that will be used to submit the form data. Possible values are "get" and "post".
enctype Spécifie l'encodage qui sera utilisé pour soumettre les données du formulaire. Les valeurs possibles sont "multipart/form-data", "application/x-www-form-urlencoded" et "text/plain"
accept Spécifie les types MIME des fichiers que l'utilisateur peut télécharger.
autocomplete Specifies whether the browser should autofill the form fields. Possible values are "on", "off", and "auto".
class Spécifie un nom de classe pour le formulaire.
id Specifies an ID for the form.
name Spécifie un nom pour le formulaire
style Spécifie le nom de l'élément d'entrée.
type Spécifie le type d'élément d'entrée. Les valeurs possibles sont "text", "password", "checkbox", "radio", "select" et "file"
name Spécifie le nom de l'élément d'entrée.
value Spécifie la valeur initiale de l'élément d'entrée.
checked Spécifie la valeur initiale de l'élément d'entrée.
value
selected Spécifie si le bouton radio est sélectionné par défaut.
size Spécifie le nombre de lignes dans une zone de texte.
cols Spécifie le nombre de colonnes dans une zone de texte.
placeholder Spécifie un indice pour l'utilisateur.
required Spécifie que l'élément d'entrée est obligatoire.
disabled Spécifie que l'élément d'entrée est désactivé.
